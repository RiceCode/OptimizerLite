API

Provide the ISBN and get the information about a sepcific book in json format.
url: http://127.0.0.1:5000/api/books/<isbn_nbr>
HTTP method: GET
Parameter: isbn_nbr
format: json
callback: function to wrap JSON response

example: http://127.0.0.1:5000/api/books/0380795272
Returns:
{
"isbn": "0380795272",
"title": "Krondor: The Betrayal",
"author": "Raymond E. Feist",
"year": 1998,
"average_score": 2.5,
"review_count": 6
}

*If there's no review, the average_score will be 0.0.
